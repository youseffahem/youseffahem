## About Me

I'm a Computer Science student at Modern Academy, Maadi (2023–2027), specializing in **Artificial Intelligence, Machine Learning, and Embedded Systems**. I design and build end-to-end engineering systems — spanning ML pipelines, real-time computer vision, and hardware-level firmware — with a strong focus on how intelligence can operate at the physical layer.

My work spans both the software and hardware domains: from training deep learning models to writing embedded C/C++ for microcontrollers, integrating sensors, and building interactive real-time systems. I'm particularly interested in the intersection of AI and embedded hardware — bringing intelligent decision-making to resource-constrained devices.

- **AI/ML Focus:** Deep Learning · NLP · Computer Vision · Generative AI
- **Embedded Focus:** Microcontroller Programming · Sensor Integration · PWM · Real-Time Systems
- **Academic Distinctions:** Numerical Analysis · Calculus · Linear Algebra
- **Open to:** Research collaborations · Internships · Open-source contributions
